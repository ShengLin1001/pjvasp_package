Examples
========

Example data and notebooks live under ``mymetal/example/``. They are not a
formal test suite; they are working research examples that show the directory
conventions used by the package.

Useful starting points
----------------------

``mymetal/example/test-generate-bulk``
   Bulk and surface model generation examples.

``mymetal/example/test-surface-energy``
   Surface-energy workflow examples.

``mymetal/example/test-stretch``
   Strained structures and batch ``y_dir`` directory examples.

``mymetal/example/test-post``
   Post-processing examples with VASP output-like directory layouts.

``mymetal/example/test-n2p2-sfparams``
   n2p2 symmetry-function and dataset examples.
