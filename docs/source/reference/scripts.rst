Script Reference
================

The repository contains several script-oriented directories:

``vasp_utils/``
   VASP workflow commands and universal submission/post-processing helpers.

``myvasp/``
   Historical VASP shell helpers and VTST-related scripts.

``slurm_utils/``
   Scheduler templates.

``lmp_utils/``
   LAMMPS input templates.

These scripts are coupled to HPC directory conventions. Review the file headers
and run them on small test directories before batch use.
